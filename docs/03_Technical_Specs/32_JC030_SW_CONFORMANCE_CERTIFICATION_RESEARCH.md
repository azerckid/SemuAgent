## 1. Purpose

JC-030 Slice 2b(fcrypt 암호화) 이전에, 더 근본적인 질문이 있다: **SemuAgent가 만든 전자신고 파일을
홈택스가 애초에 받아주는가?** [NTS Crypto Spec §4.2](./31_JC030_NTS_CRYPTO_SPEC_ACQUISITION.md)의 조사에서
"국세청은 적합성 검정을 받은 세무프로그램(약 15개 업체)이 만든 파일만 파일변환신고로 받는다"는 정황이
확인됐다. 이 문서는 그 적합성 검정 제도 자체를 조사한다.

**중요:** 이건 파일 규격(레이아웃)이나 암호화 알고리즘과 무관한 별도 게이트다. 파일을 완벽하게 만들어도
SemuAgent가 이 인증을 받지 못하면 파일변환신고 경로 자체가 막힐 수 있다.

## 5. Decision Gate

**신청 여부는 조사 완료 전까지 보류한다.** 심사 기간이 길거나 소규모 서비스 신청이 어렵다고 확인되어도,
SemuAgent의 대안은 홈택스 화면에 값을 옮겨 적도록 안내하는 방식이 아니다. Path 1은 공식 양식·전산매체 규격에 맞춘
업로드용 파일/양식 작성 지원으로 유지하고, 검정이 필요한 파일변환 경로는 Path 3로 분리한다.
