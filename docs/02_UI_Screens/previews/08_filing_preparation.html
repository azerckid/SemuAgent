책임 경계 — SemuAgent는 신고서에 넣을 확정 데이터와 제출 보조 자료를 준비합니다. 최종 제출·납부는 사용자가 홈택스·위택스에서 직접 수행합니다. 자동 제출은 JC-023의 별도 법무·보안·사용자 승인 게이트 없이는 도입하지 않습니다.
