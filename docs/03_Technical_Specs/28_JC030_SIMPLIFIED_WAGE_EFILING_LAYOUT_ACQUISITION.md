## 1. Purpose

JC-030 v1 1순위 후보는 **근로소득 간이지급명세서** 전자신고(변환) 파일이다. Slice 0a는
구현에 필요한 **공식 레이아웃 문서의 정체성·입수 경로·제출 관문**을 고정한다.

본 문서는 HWP/PDF 원문을 git에 보관하지 않는다(원칙: 원시 세무 규격서는 외부 공식
출처에서 구현 직전에 재확인). Pre-Code Brief 작성 시 최신본을 다운로드해 필드
매핑표를 만든다.

## 2. Critical Distinction — 간이지급 vs 연말 지급명세서

| 구분 | 연말 근로소득 지급명세서 | **JC-030 대상: 간이지급명세서(근로소득)** |
|---|---|---|
| 제출 주기 | 연 1회(연말정산) | **반기**(현행; 2027 지급분부터 매월) |
| JC-024 관계 | 연말정산 준비 데이터 | **반기 집계 read model** |
| 공식 레이아웃 묶음 | `2025년 귀속 지급명세서 전산매체 제출요령` | **`간이지급명세서(근로소득) 전산매체 제출요령`** |
| 홈택스 자료실 예시 | 게시번호 **1175**(2025 귀속, 2025-08-07 안내) [확실] | **1175에 포함되지 않음** [확실] |
| 서식 근거 | 근로소득 지급명세서(연말) | 소득세법 시행규칙 **별지 제24호의4서식(1)** [확실] |
| 레코드 복잡도 | A~I 레코드, 2010바이트 등(연말) | 간이 전용 레코드 세트(별도 HWP) [확실·개요] |

**판정:** bulletin 1175·`근로소득(의료비포함)_전산매체제출요령`은 JC-030 v1에 **쓰지 않는다**.
간이지급 전용 문서를 입수해야 한다.

## 3. Official Acquisition Path (Resolved)

### 3.1 1순위 — 홈택스 자료실

- URL: https://www.hometax.go.kr/ (로그인 불필요 구간) → **자료실**
- 검색 키워드: `간이지급명세서(근로소득) 전산매체 제출요령`, `간이지급명세서`, `정오표`
- 문서명(확정): **`간이지급명세서(근로소득) 전산매체 제출요령`** (+ 동반 **정오표**)
- 귀속/제출 시점: 반기별·지급분기별로 **정오표가 수시 갱신**됨 → 구현·제출 직전 최신본 필수 [확실]

### 3.2 2순위 — 국세청 누리집 국세청 프로그램 게시판

- 목록: https://www.nts.go.kr/nts/na/ntt/selectNttList.do?bbsId=1099&mi=2544
- 검색: `간이지급`
- 확인된 게시 예(구조 참고, **최신본 아님**):
  - `2019년 귀속 간이지급명세서 전산매체 제출요령` — nttSn **85418** (2019-03-28)
    - 첨부: `2019년귀속간이지급명세서4종_전산매체제출요령_20190628.hwp`, 정오표 HWP
  - `[근로소득간이지급명세서] 2019년 귀속 하반기 …` — nttSn **85087** (2020-02-25)
- **주의:** 게시판 검색 결과 최상단이 2019년일 수 있음. 최신 반기분은 **홈택스 자료실** 또는
  국세청 **공지/보도**([간이지급명세서] 제목 패턴)를 병행 확인 [확실]

### 3.3 3순위 — 국세청 공지·보도(제출 시기별)

- 패턴: `[간이지급명세서] {연도} {상·하반기|지급분} … 전산매체 제출요령`
- 예: 공지 mi=2207 nttSn=1295164 (2021 상반기, 변경 없음 안내) [확실]
- 제출 직전 해당 시기 공지에 **동일 제목 HWP**가 재게시되는 경우가 많음 [추정]

### 3.4 문의(필드·코드표 불명 시)

- 국세청 원천세과 원천2팀: **044-204-3348, 3349** (2025 연말 지급명세서 안내와 동일 연락처) [확실]
- 홈택스 고객센터 **126**

### 3.5 입수하지 않는 경로

## 4. Hometax Submission Path (변환제출)

JC-030 생성 파일의 사용자 제출 관문(자동 제출 아님):

1. 홈택스 로그인
2. **지급명세·자료·공익법인** → **일용·간이지급명세서/사업장제공자 등의 과세자료 제출명세서 제출(매월·반기)**
3. **변환 파일제출**(회계·급여 프로그램 이용)
4. (지급)명세서: **간이지급명세서(근로소득)** 선택 → 파일 업로드
5. **파일 검증**(제작 시 설정한 **전자신고 암호 8~15자리** 입력) → 형식·내용 검증 → 제출

공식 안내:

- 간이지급 제출 의무·주기: https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?cntntsId=239032&mi=40678
- 제출 방법: https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?cntntsId=239045&mi=40990
- 홈택스 바로가기(간이지급): https://www.hometax.go.kr/websquare/websquare.html?w2xPath=/ui/pp/index_pp.xml&tmIdx=44&tm2lIdx=4401000000&tm3lIdx=4401100000

직접작성 제출(홈택스 화면 입력)은 JC-013 가이드 범주이며, JC-030은 **변환제출용 파일 생성**에 초점을 둔다.

## 5. File Format Overview (Pre-Code Brief 입력용)

공식 HWP 미입수 상태에서 확정 가능한 개요만 기록한다. 필드 폭·record type 상세는
**최신 HWP 다운로드 후** Pre-Code Brief에서 확정한다.

| 항목 | 내용 | 근거 |
|---|---|---|
| 원본 레이아웃 매체 | 국세청 HWP `…전산매체 제출요령` + `정오표` | NTS 85418, CFO 2021 안내 [확실] |
| 레코드 구조 | 고정길이 record type·순서·오류검증 규칙 HWP에 정의 | 동일 [확실] |
| 배포 파일 | 급여 SW가 **fcrypt API**로 만든 암호화 전자파일; 홈택스는 **NTS-CRYPTO**로 복호화 | JC-023 §2.1, [NTS Crypto Spec](./31_JC030_NTS_CRYPTO_SPEC_ACQUISITION.md) [확실] |
| 암호 | 제작 시 **8~15자리**; 홈택스 검증 시 동일 비밀번호 | 실무·JC-023 [확실] |
| 인코딩·줄바꿈 | HWP 명시 따름 | [미확인 — HWP 필요] |
| 월별 구분 | 2022~ 급여·인정상여 **월별** 수록 | CFO 2021 하반기 안내 [확실] |
| 원천징수세액 | 간이지급명세서에는 **기재하지 않음**(소득금액·근무기간 중심) | 국세청 Q&A·세정일보 인용 [확실] |
| 소득자 식별 | 주민등록번호(또는 외국인등록번호) — 파일에 포함 | 실무·직접작성 항목 [확실]; PII는 [PII Policy](./27_JC030_EFILING_FILE_PII_POLICY.md) |
