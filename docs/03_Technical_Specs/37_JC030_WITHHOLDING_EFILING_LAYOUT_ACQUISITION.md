## 1. Purpose

Path 1은 홈택스·국세청·위택스가 제공하고 해당 사이트가 직접 수용하는 **공식
비암호화 업로드 양식**에 SemuAgent가 값을 채워 파일을 만들고, 사용자가 직접
업로드·제출하도록 돕는 경로다.

원천징수이행상황신고서에 대해서는 다음 두 사실을 별도로 검증한다.

1. 신고 내용을 작성할 공식 서식이 존재하는가.
2. 그 서식 또는 별도 파일이 암호·fcrypt·인증 도구 없이 홈택스에 직접 업로드 가능한가.

1번만 확인되고 2번이 확인되지 않으면 Path 1 파일을 만들지 않는다. HWP/PDF가
인쇄·서면 제출용이거나, 회계프로그램 변환파일이 비밀번호를 요구하면 현재 Path 1
산출물로 사용하지 않는다.

## 2. Product Decision — 2026-07-10

| 항목 | 결정 |
|:---|:---|
| Path 1a 산출물 | 공식 비암호화 업로드 양식에 값을 채운 파일 (양식 있을 때) |
| Path 1b 산출물 | 확정 데이터를 `항목 = 값`으로 정리한 직접입력 화면 (양식 없을 때) |
| 사용자 역할 | SemuAgent에서 값 확인 후 홈택스에 직접 업로드(1a) 또는 직접 입력(1b)·제출 |
| 직접입력 위치 매핑 | 포함 — 신고 메뉴 경로·화면명·기본정보·A01 행/칸을 값과 함께 표시 |
| fcrypt·비밀번호·암호화 파일 | 제외 |
| 바이너리 레코드 추정 | 금지 |
| 공식 비암호화 업로드 양식이 없는 세목 | **1b(직접입력 정리) 대상** (값 정리 화면은 별도 구현) — `blocked` 아님 |

따라서 과거 문서의 `공식 바이너리 레이아웃 입수`는 W0 완료선이 아니다.
원천세 변환프로그램·NTS-CRYPTO 조사는 Path 1 구현 입력으로 사용하지 않는다.

### 3.2 W0를 닫는 공식 원문 근거

국세청 2017 작성요령은 당시 원천세 전자신고를 다음 두 방식으로 구분한다.

1. 홈택스 화면에서 신고서를 작성해 전송하는 방식.
2. 회계프로그램에서 생성한 파일을 변환프로그램으로 형식 검증한 뒤 전송하는 방식.

같은 공식 문서는 2번 흐름에 파일 선택, 형식 검증, **비밀번호 입력** 단계를
명시한다. 이는 공개된 원천세 파일 경로가 현재 Path 1의 `공식 비암호화 양식에
값을 채워 직접 업로드` 계약과 다르다는 근거다.

이 과거 공식 안내만으로 2026년 로그인 화면에 다른 기능이 절대 없다고 단정하지 않는다.
다만 W0 통과에는 최신 공식 원본과 직접 수용의 **긍정적 증거**가 필요한데 이를 확보하지
못했고, 확인된 경로는 직접입력 또는 회계프로그램 변환파일뿐이다. 따라서 현재 구현
계약을 닫되, §5의 최신 공식 증거가 생기면 다시 연다.

## 4. W0 Final Verdict

| W0 통과 조건 | 결과 |
|:---|:---:|
| 국세청·홈택스가 제공한 비암호화 업로드 원본 파일 | 미확인 |
| 파일 형식·버전·적용일 | 미확인 |
| 정확한 직접 업로드 메뉴·버튼 | 미확인 |
| 공식 샘플의 직접 수용 | 미검증 |
| 암호·fcrypt·변환프로그램 없이 업로드 | 공식 안내와 불일치 |

**최종 판정: 공식 비암호화 업로드 양식 없음 → 원천세는 Path 1b(직접입력 안내)로 구현 완료했다.**

- W0 조사는 완료했다. 1a(양식 파일)는 공식 양식이 없어 만들지 않는다.
- W1 Part B, W2 generator, W3 API/UI, W4 업로드 검증, W5 파일 closeout은 1a 전용이며
  공식 양식이 새로 확인될 때까지 시작하지 않는다.
- 암호화·비밀번호 경로로 우회하지 않는다.
- 원천세는 홈택스 신고 경로·기본정보·A01 ④⑤⑥ 위치와 확정값을 함께 표시하는
  **1b 직접입력 안내 화면 구현이 완료됐다**. 세목을 `blocked`로 두지 않는다.

## 5. 1a Upgrade Conditions (양식 확인 시 승격)

원천세 1a(양식 파일) 트랙은 다음 자료가 새로 확보된 경우에만 시작한다. 그전에는 1b 대상으로 둔다(값 정리 화면은 구현 대기).

1. 국세청·홈택스가 배포한 최신 비암호화 원본 양식.
2. 파일 형식, 버전, 적용일과 공식 다운로드 URL.
3. 암호·fcrypt·회계프로그램 변환 없이 해당 파일을 선택하는 정확한 홈택스 메뉴.
4. 수정하지 않은 공식 샘플 또는 최소 fixture가 업로드 단계에서 수용된 기록.

홈택스 화면 작성 기능, 인쇄용 HWP/PDF, 비밀번호가 필요한 변환파일은 재개 근거가 아니다.
